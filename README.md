# satxml

Herramienta de linea de comandos para descargar archivos xml del SAT en Mexico


## Installation
``` bash
sudo make install
```
  
##Dependencias:
  openssl
  curl
  xmllint
  unzip
 
 
## Inicializar RFC
``` bash
satxml rfc RFCENMAYUSCULAS -k ruta/fiel/archivo.key -c ruta/fiel/archivo.cer -p fielpass
```
